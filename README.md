# MUMPS Code Generator & Explainer

「古くて難しいMUMPSのコードも、もう怖くない。」

本ツールは、医療・金融システムなどで利用されるMUMPS（M言語）のコード生成と解説に対応するGPTアシスタントです。

## 主な機能

- 自然言語からMUMPSコードを生成（コメント付き）
- 既存のMUMPSコードの日本語解説
- MUMPS文法や他言語との違いの説明

MUMPS初心者からレガシー保守担当者まで、幅広く活用いただけます。
